This repository is for the code that I write when completing coding challenges or working on fun side projects.

Under the package sam.ferg: 
  work done for coding challenges
Under the package sam.ferg.wordcounter:
  side project with the goal of counting the frequency of words in a text file and then generating a file which imitates the original. 
Under the package sam.ferg.fantasyfootball:
  side project with the goal of analyzing my fantasy football league. Currently capable of predicting the results of each weeks match-ups,   giving the chances of a team making the play-offs, and showing you the best results of each match-up in order for your team to make the     play-offs.
